Tuyệt Kỹ Cho Dev:
Làm Việc Từ Xa Cùng VS Code & Claude Code

Bí kíp đưa môi trường làm việc lên điện thoại hoặc bất kỳ trình duyệt nào mà không cần cấu hình VPN, chạy trực tiếp trên máy của bạn.

💡 TỔNG QUAN: Tài liệu gồm 2 phần chính:
  • Phần 1 — VS Code Remote Tunnel: Mở toàn bộ VS Code của máy Dev trên trình duyệt khác (Full IDE từ xa).
  • Phần 2 — Claude Code Remote Control: Điều khiển riêng một phiên Claude Code đang chạy, tối ưu cho điện thoại (App Claude) và có Push Notification.
Lưu ý: Cả hai đều giữ code chạy trên máy của bạn, không đẩy lên cloud — thiết bị từ xa chỉ đóng vai trò là màn hình điều khiển.

Phần 1 Remote Tunnel trên Visual Studio Code

Mục đích: Truy cập VS Code và thư mục làm việc của máy Dev từ xa qua trình duyệt tại https://vscode.dev/tunnel/<tên-tunnel>, không cần VPN.

Yêu cầu: Đã cài VS Code; Tài khoản GitHub/Microsoft; Máy Dev đang bật và mở VS Code.

1. Mở menu Account và chọn "Turn on Remote Tunnel Access..."

  • Bấm icon Account (hình người) ở góc dưới bên trái thanh Activity Bar ➔ Chọn Turn on Remote Tunnel Access...
  • Cách khác: Nhấn Ctrl+Shift+P ➔ gõ Remote Tunnels: Turn on Remote Tunnel Access.

2. Chọn cách bật

  • Hộp thoại hiện ra sẽ có 2 lựa chọn. Khuyên dùng: Turn on for this session (Tunnel chỉ chạy khi VS Code còn mở).

3. Đăng nhập tài khoản

  • Chọn Sign in with GitHub (hoặc Microsoft nếu công ty dùng). Trình duyệt sẽ tự mở để xác thực.
  • Trên trang GitHub, bấm Continue để cấp quyền cho Visual Studio Code.
  • Trình duyệt sẽ hỏi có muốn mở lại VS Code không, bấm Open Visual Studio Code để quay lại App.

4. Lấy đường link truy cập từ xa

  • Quay lại VS Code, mở panel OUTPUT và chọn channel Remote Tunnel Service ở góc phải.
  • Log sẽ báo tunnel đã chạy kèm tên tunnel. Copy đường link (vd: https://vscode.dev/tunnel/w010955143).

5. Truy cập và làm việc từ xa

  • Trên máy hoặc trình duyệt khác, mở đường link vừa copy.
  • VS Code Web sẽ hiện ra. Bấm Open Folder, nhập đường dẫn thư mục dự án trên máy Dev (vd: C:\Users\<user>\...) rồi bấm OK để bắt đầu làm việc.
⚠️ Lưu ý bảo mật (Phần 1):
  • Chỉ bật khi cần, tắt khi xong (Account ➔ Turn off Remote Tunnel Access).
  • Nên dùng tài khoản công ty có bật 2FA.
  • Không bật trên máy Production hoặc với dữ liệu nhạy cảm chưa được duyệt.

Phần 2 Remote Control session của Claude Code

Mục đích: Đang code dở một phiên Claude Code trên máy, bạn có thể cầm điện thoại hoặc mở trình duyệt khác để tiếp tục điều khiển chính phiên đó.

Yêu cầu: Gói Claude Pro/Max/Team/Enterprise; Claude Code CLI v2.1.51+ (kiểm tra bằng claude --version).

1. Kích hoạt tính năng

  • Trong ô prompt của Claude Code (ngay trong Terminal của VS Code), gõ lệnh /remote-control (hoặc /rc) rồi nhấn Enter.
  • Một Banner sẽ hiện ra phía trên báo trạng thái kết nối, kèm theo Link phiên (Session link).

2. Mở phiên trên thiết bị khác

  • Bấm Open in browser trên banner.
  • Hoặc trên điện thoại: Mở app Claude ➔ Tab Code ➔ Chọn phiên. (Gõ /mobile trong Claude Code để hiện QR Code tải App).

3. Điều khiển từ xa & Bật thông báo đẩy

  • Bây giờ bạn có thể gửi tin nhắn, duyệt thay đổi code từ điện thoại. Mọi thứ đồng bộ realtime.
  • Để nhận thông báo về điện thoại: Mở App Claude ➔ Terminal chạy lệnh /config ➔ Bật Push when Claude decides.
  • Khi làm xong, bấm icon Close trên banner hoặc chạy lại lệnh /remote-control để ngắt kết nối.
⚠️ Mẹo & Lưu ý (Phần 2):
  • Bảo mật: Chỉ kết nối ra ngoài bằng HTTPS, không mở cổng (port) vào máy. Tuy nhiên, ai có link phiên là điều khiển được ➔ Hãy giữ link cẩn thận như mật khẩu!
  • Claude vẫn chạy trên máy của bạn. Đóng VS Code/terminal thì phiên cũng sẽ tự kết thúc.
  • Muốn hiện mã QR nhanh để quét: Chạy lệnh claude remote-control "Tên dự án" trên CLI bên ngoài (không dùng trong VS Code) rồi nhấn dấu cách (Spacebar).
