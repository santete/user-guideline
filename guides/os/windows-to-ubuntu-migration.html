Cẩm Nang Sinh Tồn:
Chuyển Đổi Từ Windows Sang Ubuntu

Bí kíp giúp dân Văn phòng, IT, Design, Data chuyển nhà sang Linux một cách mượt mà nhất trước làn sóng rà soát bản quyền phần mềm.

💡 BỐI CẢNH: Hiện nay, việc rà soát và kiểm duyệt bản quyền phần mềm (Windows, Microsoft Office, Adobe...) tại các doanh nghiệp diễn ra rất gắt gao. Chuyển sang sử dụng hệ điều hành mã nguồn mở (Ubuntu) là giải pháp An Toàn, Hợp Pháp, Miễn Phí và Tối Ưu nhất.

1 Ánh xạ phần mềm: Không lo "mất đồ nghề"

Dù bạn ở vị trí nào, thế giới Open Source luôn có phần mềm tương đương để bạn làm việc:

  • 👔 Khối Văn phòng (Back Office): Sử dụng WPS Office hoặc LibreOffice. Giao diện và cách dùng giống Microsoft Word, Excel, PowerPoint đến 99%.
  • 💻 Khối Developer: Ubuntu là thiên đường! Mọi công cụ như VS Code, Docker, Git, IntelliJ, DBeaver đều hỗ trợ Native, chạy nhanh và mượt hơn Windows.
  • 🎨 Khối Design: Sử dụng Figma (trên trình duyệt), GIMP (thay Photoshop), Inkscape (thay Illustrator).
  • 📊 Khối Data / Business Analytics: Dễ dàng cài đặt Python, Anaconda, TablePlus, Metabase, PowerBI (qua Web).

2 Đồng bộ Dữ liệu Trình duyệt (Rất quan trọng)

Trước khi chia tay Windows, bạn cần đẩy toàn bộ Bookmark (Dấu trang), Mật khẩu và Lịch sử duyệt web lên Mây (Cloud).

  • Trên máy Windows: Mở Google Chrome, bấm vào biểu tượng Avatar ở góc trên cùng bên phải. Bấm "Bật tính năng đồng bộ hóa" (Turn on sync...) và đăng nhập bằng tài khoản Gmail của bạn.
  • Trên máy Ubuntu: Sau khi cài Chrome, bạn chỉ cần đăng nhập lại Gmail đó. Mọi thứ sẽ tự động tràn về như cũ!

3 Cấu hình Proxy và Local Hosts (Dành cho mạng Công ty)

Ở một số công ty, bạn cần cấu hình Proxy để ra Internet và sửa file Hosts để vào các trang Web nội bộ.

A. Xử lý Proxy:

  • Lấy Proxy ở Windows: Mở Start Menu > gõ Internet Options > tab Connections > LAN settings. Ghi lại địa chỉ Proxy và Port.
  • Cài Proxy ở Ubuntu: Vào Settings > Network > Network Proxy. Bấm nút bánh răng, chọn Manual và điền lại đúng thông số Proxy & Port bạn vừa ghi.

B. Backup và chuyển file Hosts:

Tại sao cần file Hosts? File Hosts chứa các địa chỉ IP trỏ vào Tên miền nội bộ (Local) chưa được Public ra ngoài. Không có nó, bạn sẽ không thể truy cập các Tool nội bộ của công ty!
  • Bước 1 (Trên Windows): Truy cập thư mục C:\Windows\System32\drivers\etc\. Mở file có tên là hosts bằng Notepad, Copy toàn bộ nội dung và lưu lên Google Drive/Zalo truyền file.
  • Bước 2 (Trên Ubuntu): Mở Terminal (phím tắt Ctrl + Alt + T) và gõ lệnh sau để mở file hosts:
    sudo nano /etc/hosts
  • Bước 3: Nhập mật khẩu máy tính của bạn. Sau đó Dán (Paste) toàn bộ nội dung đã copy từ Windows vào phía dưới cùng.
  • Bước 4: Bấm Ctrl + O rồi Enter để Lưu, sau đó bấm Ctrl + X để thoát. Hoàn tất!

4 Cài đặt các Tiện ích "Bắt buộc phải có"

1. Bộ gõ Tiếng Việt (ibus-bamboo)

Đây là bộ gõ tiếng Việt tốt nhất trên Linux hiện tại, trải nghiệm y hệt Unikey.

  • Mở Terminal và chạy lần lượt các lệnh sau:
sudo add-apt-repository ppa:bamboo-engine/ibus-bamboo
sudo apt-get update
sudo apt-get install ibus-bamboo
ibus restart
  • Vào Settings > Region & Language > Input Sources, thêm tiếng Việt (Vietnamese - Bamboo).

2. Công cụ chụp ảnh màn hình & Ghi chú (Flameshot)

Zalo hay Lightshot trên Windows cho phép bạn chụp màn hình, vẽ mũi tên, ghi chú text ngay lập tức. Trên Ubuntu, Flameshot là ông vua của tính năng này.

  • Cài đặt qua Terminal: sudo apt install flameshot
  • Cài phím tắt Print Screen (PrtSc) để gọi Flameshot:
  • Vào Settings > Keyboard > Keyboard Shortcuts > Custom Shortcuts.
  • Thêm mới: Name là Flameshot, Command là flameshot gui, Shortcut là phím PrtSc.

5 Cài đặt Phần mềm Giao tiếp & Văn phòng

  • Trình duyệt Google Chrome: Tải trực tiếp file .deb từ trang chủ Google và nháy đúp để cài đặt.
  • WPS Office: Thay thế hoàn hảo cho Word, Excel. Tải file .deb từ trang chủ linux.wps.com.
  • Zalo / Telegram / Microsoft Teams: Với Telegram bạn có thể cài trực tiếp từ Ubuntu Software (App Store của Ubuntu). Với Zalo và Teams, khuyên dùng bản Web (có thể cài thành PWA Web App bằng cách vào Chrome > Dấu 3 chấm > Save and Share > Install page as app).
