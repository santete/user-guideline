Hướng Dẫn Chi Tiết:
Đọc Tài Liệu Markdown (.md) Trực Tiếp Trên Google Chrome

Tài liệu này giúp bạn dễ dàng mở, đọc và điều hướng các file .md với giao diện chuyên nghiệp, mục lục rõ ràng và đảm bảo chữ không bao giờ bị đè.

1 Cài đặt công cụ đọc tài liệu (Markdown Viewer)

Chúng ta sẽ sử dụng công cụ Markdown Viewer. Đây là công cụ tốt nhất, miễn phí và có giao diện chia cột rất gọn gàng.

  • Bước 1: Mở trình duyệt Google Chrome trên máy tính.
  • Bước 2: Nhấn vào link này: Cửa hàng Chrome trực tuyến, sau đó gõ tìm từ khóa Markdown Viewer.
  • Bước 3: Bấm nút Thêm vào Chrome (Add to Chrome) màu xanh, sau đó một bảng hỏi hiện ra, bạn bấm tiếp Thêm tiện ích.

2 Cấp quyền cho Chrome đọc file từ máy tính

⚠️ BƯỚC BẮT BUỘC: Vì tài liệu nằm sẵn trên máy tính, Chrome mặc định sẽ chặn không cho mở. Bạn phải làm bước này thì công cụ mới hoạt động!
  • Bước 1: Nhấn vào link sau để mở trang Quản lý tiện ích: 🧩 chrome://extensions/ (Nếu click không được, hãy copy dán vào thanh địa chỉ).
  • Bước 2: Tìm đến ô của Markdown Viewer và bấm vào nút Chi tiết (Details).
  • Bước 3: Cuộn chuột xuống dưới, tìm dòng Cho phép truy cập vào URL của tệp (Allow access to file URLs) và bật công tắc sang màu xanh.

3 Cấu hình giao diện và bật Mục lục cố định (Không đè chữ)

Bước này giúp tài liệu hiển thị đẹp như sách và có một cột mục lục bên cạnh để chuyển chương nhanh chóng.

  • Bước 1: Bấm chuột phải vào biểu tượng của Markdown Viewer ở góc trên cùng bên phải trình duyệt, chọn Tùy chọn (Options / Settings).
  • Bước 2: Tại cột Theme, ở mục Content Theme hãy chọn GITHUB để có nền trắng, phông chữ to rõ dễ đọc nhất.
  • Bước 3: Tại cột Content, tìm mục toc (Table of Contents) và bật công tắc sang màu xanh (ON).
💡 Cài đặt xong, mục lục sẽ tự động hiển thị gọn gàng ở một cột riêng biệt, nội dung tài liệu sẽ co lại cho vừa vặn. Khắc phục triệt để hiện tượng mục lục che mất chữ thường gặp!

4 Cách mở file lần đầu tiên cực dễ: Kéo và Thả

Bạn không cần tìm menu mở file lằng nhằng, chỉ cần "Kéo và Thả".

  • Mở sẵn một trang Chrome trắng.
  • Mở thư mục chứa file .md trên máy tính.
  • Bấm giữ chuột trái vào file .md, kéo nó sang khoảng trắng giữa màn hình Chrome rồi thả tay ra. File sẽ hiện lên với bố cục chia cột tuyệt đẹp!

5 Tùy chọn: Cài đặt nháy đúp mở file tự động cho lần sau

Để từ nay về sau, cứ nháy đúp chuột (mở bình thường) là file nhảy lên Chrome:

👉 Máy Windows:

  • Click chuột phải vào file .md ➔ Open with ➔ Choose another app.
  • Chọn Google Chrome.
  • Đánh dấu tích vào ô Always use this app to open .md files và bấm OK.

👉 Máy Mac (Apple):

  • Click chuột phải (hoặc nhấn 2 ngón) vào file ➔ Get Info.
  • Tìm mục Open with: và chọn Google Chrome.
  • Bấm tiếp nút Change All... bên dưới ➔ Continue để xác nhận.
